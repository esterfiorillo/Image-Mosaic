# Image-Mosaic

Algorithm that combines two images that have an intersection into a single mosaic image.

Example:

Image 1: 

![Image1](img1.jpg)

Image 2:

![Image2](img2.jpg)

Resultant Mosaic:

![Mosaic](mosaic.png)

How to execute it:

```
python3 main.py --file1 img1.jpg --file2 img2.jpg
```
